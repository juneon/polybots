## 상황별 안내

- **세션 재개** → `WORKLOG.md` 상단(결정 표·로드맵) + 마지막 진행 로그부터
- **새 전략 추가** → `CLAUDE.md` "새 전략 추가" 절차 (SPEC §6 문서화 + tests/ 추가 포함)
- **백테스트/재평가** → `backtest/README.md` (또는 UI Backtest 탭)
- **live 관련 판단** → `CLAUDE.md` 안전 규칙 + `WORKLOG.md` 로드맵의 "live 재개 기준"
